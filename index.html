box-sizing: content-boxWidth = content only. Padding & border added on top.

        
box-sizing: border-boxWidth includes padding & border. Recommended default.

        
margin: autoCenters a block element horizontally.

        
padding: 1rem 2remShorthand: top/bottom, left/right.

        
border: 1px solid #cccWidth, style, color shorthand.

        
outlineLike border but doesn't affect layout.

        
overflow: hiddenClips content, clears floats.

        
visibility: hiddenHidden but still takes up space.

        
display: noneRemoved from layout entirely.

      

    

  

  

    

    
/* Universal border-box reset (recommended) */
*, *::before, *::after {
  box-sizing: border-box;
}
.box {
  width: 300px;
  padding: 1rem 1.5rem;
  border: 2px solid #7c6aff;
  margin: 0 auto;
  /* Total width = 300px (border-box) */
}

    08
    
Positioning & Z-Index

  

  

    
static
Default. Follows normal document flow. top/left/etc. have no effect.

    
relative
Stays in flow, but top/left offset it from its original position. Creates stacking context.

    
absolute
Removed from flow. Positioned relative to nearest non-static ancestor.

    
fixed
Removed from flow. Positioned relative to viewport. Stays on scroll.

    
sticky
Relative until scroll threshold, then acts like fixed. Needs overflow: scroll on ancestor.

    
z-index
Controls stacking order. Only works on positioned or flex/grid items. Higher = on top.

  

  

    
Common z-index scale

    

      
Base contentz-index: 0

      
Dropdown / Popoverz-index: 100

      
Sticky headerz-index: 200

      
Modal overlayz-index: 300

      
Toast / Notificationz-index: 400

CSS Units

  

  

    

      
 Absolute & Relative

      
pxPixels. Fixed. Screen baseline.border: 1px

      
remRoot em. Relative to <html> font-size (16px).padding: 1rem

      
emRelative to parent element's font-size.margin: 1.5em

      
%Percentage of parent's dimension.width: 50%

      
vw1% of viewport width.font-size: 5vw

      
vh1% of viewport height.height: 100vh

      
dvhDynamic viewport height. Accounts for mobile browser chrome.height: 100dvh

      
chWidth of the "0" glyph in current font.max-width: 65ch

      
frFraction unit. Grid only.1fr 2fr 1fr

      
clampmin / preferred / max fluid range.clamp(1rem,5vw,3rem)

    

    

      

        
 When to use what

        

          
Font sizesrem (accessible, scales with user preference)

          
Spacing / paddingrem or px for small values

          
Component widths% or ch (content-relative)

          
Full-screen sections100dvh (mobile-safe)

          
Grid columnsfr + minmax()

          
Borders / shadowspx (stay sharp at any zoom)

          
Responsive headingsclamp()

        

      

      

        

        
/* 1rem = 10px trick */
html { font-size: 62.5%; }
body { font-size: 1.6rem; } /* = 16px */

/* Now: 1.4rem = 14px, 2rem = 20px */
h1   { font-size: 3.6rem; } /* 36px */
h2   { font-size: 2.4rem; } /* 24px */

CSS Custom Properties

  

  

    

      

      
/* Define on :root for global scope */
:root {
  --color-primary:   #7c6aff;
  --color-surface:   #131318;
  --space-sm:        0.5rem;
  --space-md:        1rem;
  --radius:          8px;
  --transition:      all 0.2s ease;
}

/* Use with var() */
.button {
  background: var(--color-primary);
  border-radius: var(--radius);
  padding: var(--space-sm) var(--space-md);
  transition: var(--transition);
}

/* Fallback value */
color: var(--missing, #ff6b6b);

/* Override in local scope */
.dark-section {
  --color-primary: #a89bff;
}

    

    

      

        

        
// Read a CSS variable
const val = getComputedStyle(document.documentElement)
  .getPropertyValue('--color-primary');

// Set a CSS variable
document.documentElement.style
  .setProperty('--color-primary', '#ff6b6b');

// Theme toggle
const toggle = () => {
  document.documentElement
    .classList.toggle('dark');
};

CSS Grid Reference

  

  

    

      
 Container Properties

      

        
display: gridActivates grid layout

        
grid-template-columnsColumn track sizes

        
grid-template-rowsRow track sizes

        
grid-template-areasNamed area layout

        
gap / column-gap / row-gapGutter sizes

        
repeat(3, 1fr)3 equal columns

        
minmax(200px, 1fr)Min 200px, grows to fill

        
auto-fill vs auto-fitauto-fill keeps empty tracks; auto-fit collapses them

        
justify-items / align-itemsCell content alignment

        
place-items: centerShorthand for justify + align

      

    

    

      
 Item Properties

      

        
grid-column: 1 / 3Span from line 1 to 3

        
grid-column: span 2Span 2 columns

        
grid-row: 1 / -1Full height (first to last line)

        
grid-area: headerReference a named area

        
justify-selfOverride cell horizontal alignment

        
align-selfOverride cell vertical alignment

        
place-self: centerCenter item in cell

      

      

      
 Responsive auto-fit pattern

      
.grid {
  display: grid;
  grid-template-columns:
    repeat(auto-fit, minmax(240px, 1fr));
  gap: 1rem;
}
